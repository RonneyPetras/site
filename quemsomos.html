Quem Somos

O "nome do jogo/site" é um projeto constituído por Ariel Vieira, Eduardo Rodrigues, Flavio Santos, Gustavo Henrique, e por Rooney Petras

O projeto consiste no desenvolvimento de uma aplicação web com a finalidade de aplicar e melhorar os conhecimentos que os alunos estão aprendendo no curso de Sistemas para Internet no IFTM
